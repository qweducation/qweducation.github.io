A. If the letters and numbers can be repeated in the same plate number, then there are 26^3 (26*26*26) possible arrangements for the letters, and 10^3 (10*10*10) possible arrangements for the numbers. Therefore, the total number of possible plate numbers is:

26^3 * 10^3 = 17,576,000

B. If the letters and numbers cannot be repeated in the same plate number, then there are 26*25*24 possible arrangements for the letters (since the first letter can be any of the 26 letters, the second letter can be any of the remaining 25, and the third can be any of the remaining 24), and 10*9*8 possible arrangements for the numbers. 26*25*24 *10*9*8 = 11,232,000
