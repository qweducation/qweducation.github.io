2. 15 kilograms of sugar cost Php 427.50.

Php 28.50 x 15 kg of sugar = Php 427.50

2 + 3 = 5

20 ÷ 5 = 4

4 x 2 = 8

4 x 3 = 12

12 + 8 = 20 balloons

3 + 4 + 5 = 12

Php 3,000 ÷ 12 = Php 250.00

Php 250 x 3 = Php 750.00

Php 250 x 4 = Php 1,000.00

Php 250 x 5 = Php 1,250.00

Php 750 + Php 1,000 + Php 1,250 = Php 3,000.00

21. Kompyutin ang GNP/GNI sa paraang Final Expenditure Approach. Ibigay ang pormulang ginamit. GK = P201 M X = P40 M GP= P310 M NPIA P52 M Ilan ang GNP/GNI? Tiyakin 1. 2. GG = P252 M SD = P2 M M-P52 M Ipaliwanag ang sagot. Bakit mahalagang malaman ang gastusin ng bawat sektor ng bansa?​


Ang formula ng GNP/GNI gamit ang Final Expenditure Approach ay:

GNP/GNI = C + I + G + (X-M)

Kung susubstitute natin ang mga halaga:

GNP/GNI = P201M + P40M + P252M + (P52M - P52M)
GNP/GNI = P441M

Ang GNP/GNI ay P441M.

Mahalagang malaman ang gastusin ng bawat sektor ng bansa dahil ito ay nagpapakita kung saan napupunta ang mga pondo at kung alin ang mga sektor na nagbibigay ng pinakamalaking kontribusyon sa ekonomiya. Ito ay nakatutulong sa pagpaplano ng mga hakbang ng pamahalaan upang mapalakas pa ang mga sektor na ito at maiangat ang kabuhayan ng mga mamamayan.
