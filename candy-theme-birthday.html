Answer:

24

Step-by-step explanation:

We can use algebra to solve the problem. Let's denote the total number of candies Li Xin had at first as "x".

Li Xin gave some candies equally to 15 friends and had 6 candies left, so the number of candies she gave away is equal to (15 × candies per friend) + 6. We don't know the exact number of candies per friend, so we can represent it with "y".

Therefore, we have the equation:

x = 15y + 6

We also know that each friend received the same number of candies, so we can divide the total number of candies equally among the 15 friends:

y = (x - 6) / 15

Now we can substitute the second equation into the first equation and solve for x:

x = 15y + 6

x = 15[(x - 6) / 15] + 6

x = x - 6 + 6

x = x

We end up with the equation x = x, which means that any value of x satisfies the equation. However, we're looking for the initial number of candies Li Xin had, which must be a positive integer.

Since she gave away some candies and had 6 left, we know that x is at least 21 (6 + 15). We can try some values of x greater than or equal to 21 and check if they work:

If x = 21, then y = (21 - 6) / 15 = 1, so Li Xin gave away 15 candies and had 6 left, which works.

If x = 22, then y = (22 - 6) / 15 ≈ 1.067, which is not a whole number, so it doesn't work.

If x = 23, then y = (23 - 6) / 15 ≈ 1.133, which is not a whole number, so it doesn't work.

If x = 24, then y = (24 - 6) / 15 = 1.2, which is a whole number, so Li Xin gave away 18 candies and had 6 left, which works.

Therefore, the initial number of candies Li Xin had is 24.
