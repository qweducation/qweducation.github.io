26. Find the [H30+] and the [OH-] from the following values. 1. pH=2.5 2. pH=11.3 3. pOH=4.6 4. pOH=8.7 5. pH=7.65 6. pH=6.0


Answer:

Explanation:

To find the [H3O+] and the [OH-] from the given values, we will use the following equations:

pH = -log[H3O+]

pOH = -log[OH-]

pH + pOH = 14

pH=2.5

[H3O+] = 10^(-pH) = 10^(-2.5) = 3.16 x 10^(-3) M

[OH-] = 1.0 x 10^(-14) / [H3O+] = 1.0 x 10^(-14) / 3.16 x 10^(-3) = 3.16 x 10^(-12) M

pH=11.3

[H3O+] = 10^(-pH) = 10^(-11.3) = 5.01 x 10^(-12) M

[OH-] = 1.0 x 10^(-14) / [H3O+] = 1.0 x 10^(-14) / 5.01 x 10^(-12) = 1.99 x 10^(-3) M

pOH=4.6

[OH-] = 10^(-pOH) = 10^(-4.6) = 2.51 x 10^(-5) M

[H3O+] = 1.0 x 10^(-14) / [OH-] = 1.0 x 10^(-14) / 2.51 x 10^(-5) = 3.98 x 10^(-10) M

pOH=8.7

[OH-] = 10^(-pOH) = 10^(-8.7) = 1.87 x 10^(-9) M

[H3O+] = 1.0 x 10^(-14) / [OH-] = 1.0 x 10^(-14) / 1.87 x 10^(-9) = 5.35 x 10^(-6) M

pH=7.65

[H3O+] = 10^(-pH) = 10^(-7.65) = 2.19 x 10^(-8) M

[OH-] = 1.0 x 10^(-14) / [H3O+] = 1.0 x 10^(-14) / 2.19 x 10^(-8) = 4.57 x 10^(-7) M

pH=6.0

[H3O+] = 10^(-pH) = 10^(-6.0) = 1.0 x 10^(-6) M

[OH-] = 1.0 x 10^(-14) / [H3O+] = 1.0 x 10^(-14) / 1.0 x 10^(-6) = 1.0 x 10^(-8) M
