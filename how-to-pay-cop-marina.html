26. marina can do an art project in 5 days. If Ruben decides to work togethet with Marina, it will take them 3 days to finish the project. How long can Ruben finish the art project alone?​


Answer:

7  ¹/₂  days

Step-by-step explanation:

Analyze:

The job, finishing the art project:  1Marina's work rate:  1/5Working together with Ruben, the combined work rate: 1/3Ruben's work rate:  1/x

Where x is the number of days Ruben can finish the art project alone.

Note that work rate is an indirect variation or inverse proportion where when more people work together to finish the same job, it will be faster or will take them fewer days to finish the same.  When fewer people or individual work on the the same job, it will take more days to complete it.

Working together ⇒  Addition

Equation:

1/5  +  1/x  = 1/3

LCD (least common denominator):  (5)(x)(3) = 15x

Multiply each term by 15x:

(1/5)(15x)  +  (1/x)(15x)  =  (1/3)(15x)

3x + 15 = 5x

5x - 3x = 15

2x/2 = 15/2

x = 15/2  or  7 ¹/₂  days

It takes 7  ¹/₂ days for Ruben to finish the art project when working alone.

Check:

x = 15/2

1/5  +  1/x  = 1/3

1/5  +  1/(15/2) = 1/3

1/5 + 2/15 = 1/3  

When dividing fraction, multiply the numerator by the reciprocal of denominator,  1/(15/2)  ⇒  1 (2/15) = 2/15

LCD:  15  hence,  1/5 = 3/15

3/15 + 2/15 = 1/3

5/15 = 1/3

1/3 = 1/3  (True)
