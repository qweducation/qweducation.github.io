6. What is the item in the color coding scheme that allows the color-coded value to deviate to a certain extent


Answer:

The item in the color coding scheme that allows the color-coded value to deviate to a certain extent is the tolerance.

In resistors, most of the time there are four (4) lines which has its corresponding value.

The first 2 lines represent the first 2 digits of the value. The third color represents the multiplier. And the last color represents its tolerance.

For example: Red Red Red Gold

Where red represents 2 which means the first two colors represent 22.The third line, being a multiplier, represents a multiple of 100.Which means that the resistor above has a value of 2200 ohms.Gold represents 5% tolerance.This means that 2200 ohms can be between 2,090 up to 2,310.

16. what is resistor color coding?


Answer:

An electronic color code is used to indicate the values or ratings of electronic components, usually for resistors, but also for capacitors, inductors, diodes and others. A separate code, the 25-pair color code, is used to identify wires in some telecommunications cables.
