1. Two forces act on the hook. Determine the magnitude of the resultant force.​ Need help for Engineering Mechanics.


Answer:

R = 665.739 N

Step-by-step explanation:

F₁ = 200 N

θ₁ = 30°

F₂ = 500 N

θ₂ = 30° + 40° = 70°

F₁x = 200cos(30°) = 173.205 N

F₁y = 200sin(30°) = 100 N

F₂x = 500cos(70°) = 171.01 N

F₂y = 500sin(70°) = 469.846 N

Rx = ΣFx

Rx = 173.205 + 171.01

Rx = 344.215 N

Ry = ΣFy

Ry = 100 + 469.846

Ry = 569.846 N

[tex]\Large{\mathtt{R=\sqrt{(Rx)^{2}+(Ry)^{2}}}}[/tex]

[tex]\Large{\mathtt{R=\sqrt{(344.215)^{2}+(569.846)^{2}}}}[/tex]

[tex]\Large{\boxed{\texttt{R = 665.739 N}}}[/tex]


2. Determine the required magnitude of force F, if the resultant couple moment on the beam is to be zero. Need help for Engineering mechanics.​


Answer: F=14.2kN•m

Step-by-step explanation:

M = (FSin30° × 3m) - (FCos30° × (0.5m + 0.5m)) = 0.634F

= 5kN(3/5)(0.5m + 0.5m) - 5kN(4/5)(3) = -9kN•m

M(sub)R = 0

F = 9kN•m/0.634 = 14.196 or 14.2kN•m
