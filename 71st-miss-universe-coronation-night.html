13. the 3rd term of an arithmitic sequence is 8 and the 16th term is 47. friend d, a, and the 71st term.​


Answer:

d = 3

a = 2

a71 = 212

How to get d:

a14 = a1 + (n-1)d

47 = 8 + (14-1)d

47 = 8 + 13d

47 - 8 = 13d

39 = 13d

39/13 = 13d/13

d = 3

How to get a:

a3 = a1 + (n-1)d

8 = a1 + (3-1)3

8 = a1 + (2)3

8 = a1 + 6

8 - 6 = a1

a1 = 2

How to get the 71st term:

a71 = a1 + (n-1)d

a71 = 2 + (71-1)3

a71 = 2 + (70)3

a71 = 2 + 210

Answer:

434

Step-by-step explanation:

In an arithmetic sequence, the difference between consecutive terms is always the same. In this case, the difference between the first two terms, 14 and 20, is 6, so we know that this is the common difference of the sequence.

To find the 71st term of the sequence, we need to start with the first term, 14, and add the common difference, 6, a total of 70 times. This can be written as 14 + 70*6 = 14 + 420 = 434. Therefore, the 71st term of the sequence is 434.

We can also use the formula for the nth term of an arithmetic sequence, which is a_n = a_1 + (n-1)*d, where a_1 is the first term, n is the term number, and d is the common difference. In this case, we can plug in the values to get a_71 = 14 + (71-1)6 = 14 + 706 = 434, which is the same result as we found using the previous method.

An arithmetic sequence is a sequence of numbers where each term is obtained by adding a fixed number, called the common difference, to the previous term. In this case, the common difference is 6, since the difference between consecutive terms is 6: 20 - 14 = 6, 26 - 20 = 6, 32 - 26 = 6, and so on.

To find the 71st term of the sequence, we need to find the first term, which is 14, and then add 70 times the common difference of 6. Therefore, the 71st term of the sequence is 14 + 70 * 6 = 14 + 420 = 434.
