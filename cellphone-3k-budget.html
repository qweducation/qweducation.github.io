2. Solve for " k " 3k + 3k = 6


Answer

[tex]\large\textrm{ x = 1}[/tex]

Further explanation

All equations aim for the same thing - isolating the variable.

When given an equation, we notice that a few operations have been performed to make the equation. So to find x we do the operations backwards. We keep working like that until x (or any other variable in an equation) is left all alone on one side of the equation, which is usually the left side.

Now, we begin solving this equation by combining the like terms:

[tex]\large\textrm{3k + 3k = 6}[/tex]

[tex]\large\textrm{6k = 6}[/tex]

Divide each side by 6:

[tex]\large\textrm{x = 1}[/tex]

Therefore, x = 1.

Notice that this equation required only two operations. There are equations that require more than that.
