26. (Answer with solutions) A financial advicor called five annual premiums on new policies as follows: 150,000,00 php 120,000,00 php 200,000,00 php 95,000,00 php and 300,000,00 php oftje insurance company gives a 30% commission for these premiums, how much will be the total commissions of the Advisor?


To find the total commission of the advisor, we need to add up the commissions for each premium. The commission for each premium is 30% of the premium amount:

Commission = 0.3 x Premium

So for the five premiums given, the commissions are:

Commission 1 = 0.3 x 150,000,00 php = 45,000,00 php

Commission 2 = 0.3 x 120,000,00 php = 36,000,00 php

Commission 3 = 0.3 x 200,000,00 php = 60,000,00 php

Commission 4 = 0.3 x 95,000,00 php = 28,500,00 php

Commission 5 = 0.3 x 300,000,00 php = 90,000,00 php

Total commission = Commission 1 + Commission 2 + Commission 3 + Commission 4 + Commission 5

Total commission = 45,000,00 php + 36,000,00 php + 60,000,00 php + 28,500,00 php + 90,000,00 php

Total commission = 259,500,00 php

Therefore, the total commission of the advisor is 259,500,00 php.
