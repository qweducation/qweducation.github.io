Step-by-step explanation:

a1 = 2.5

Formula for arithmetic sequence:

an=a1 +(n-1)d

Substitute.

23.5 = 2.5 + (n-1) 1.5

23.5 = 2.5 + 1.5n - 1.5

Rearrange.

23.5 = 2.5 - 1.5 + 1.5n

Subtract.

23.5 = 1.0 + 1.5n

Transpose.

23.5 - 1.0 = 1.5n

Subtract.

22.5 = 1.5n

Divide both sides by 1.5.

[tex]\frac{22.5}{1.5}[/tex] = [tex]\frac{1.5}{1.5}[/tex]

Cancel 1.5 on the right side then answer 22.5 divided by 1.5.

15=n

To check,

an = a1 + (n-1) d

a15 = 2.5 + (15-1) 1.5

a15 = 2.5 + (14) 1.5

a15 = 23.5

10. Given: a1 = 7 an = 25 d = 3 Find the number of terms.


[tex]\tt{\huge{\blue{Explanation:}}}[/tex]

The nth term of an arithmetic sequence is

[tex]\boxed{a_{n} = a_{1} + (n - 1)(d)}[/tex]

where:

aₙ = nth term

a₁ = first term

n = number of terms

d = common difference

[tex]\tt{\huge{\red{Solution:}}}[/tex]

Solving for the number of terms

[tex]a_{n} = a_{1} + (n - 1)(d)[/tex]

[tex]25 = 7 + (n - 1)(3)[/tex]

[tex]3(n - 1) = 25 - 7[/tex]

[tex]3(n - 1) = 18[/tex]

[tex](n - 1) = 6[/tex]

[tex]n = 6 + 1[/tex]

[tex]\boxed{n = 7}[/tex]

Therefore, the number of terms is 7.

SUM OF AN ARITHMETIC SEQUENCE

Find the sum of the first 22 natural numbers.

[tex]\bold{GIVEN:}[/tex]

[tex]a_1[/tex] = 1 [tex]n[/tex] = 22 [tex]a_n[/tex] = 22 [tex]S_n[/tex] = ?

[tex]\bold{SOLUTION:}[/tex]

Since the first and last term of the sequence are given, we will use the formula [tex]S_n=\frac{n}{2}(a_1+a_n)[/tex] where:

[tex]S_n[/tex] = sum of the first n terms[tex]a_1[/tex] = first term [tex]a_n[/tex] = last term

[tex] \begin{array}{l} \tt S_n = \frac{n}{2} (a_1 + a_n) \\ \\ \tt S_n = \frac{22}{2} (1 + 22) \\ \\ \tt S_n = 11(23) \\ \\ \red{ \boxed{ \tt S_n = 253}} \end{array}[/tex]

[tex]\\[/tex]

∴ The sum of the first 22 natural numbers is 253.

[tex]S_n=\frac{n}{2}(a_n+a_1)\\S_{48}=\frac{48}{2}(-95-1)\\S_{48}=24(-96)\\\boxed{S_{48}=-2,304}[/tex]

[tex]S_n=\frac{n}{2}(a_n+a_1)\\S_{35}=\frac{35}{2}(-95+7)\\S_{35}=\frac{35}{2}(-88)\\S_{35}=35(-44)\\\boxed{S_{35}=-1,540}[/tex]
