2. code that signals the start of protien synthesis is?


The process of converting the information in messenger RNA into a sequence of amino acids that make a protein is known as translation. As translation begins, mRNA binds to a ribosome. Then, tRNA molecules, each carrying a specific amino acid, approach the ribosome. The tRNA anticodon pairs with the first mRNA (start) codon arginine-uracil-guanine (AUG), to form the initiation complex. The two molecules temporarily join together. Usually, the first codon on mRNA is AUG, which codes for the amino acid methionine. AUG signals the start of protein synthesis. Then, the ribosome slides along the mRNA to the next codon.

19. When the programmers started writing the code, the customer/client keeps on changing and adding more requirements. This cause is under?


This is under the Step 1 phase of the System Development called "Identification of Requirements or Requirement Gathering".

Requirement Gathering is a phase for System Development wherein system developers and programmers visits the company and make a survey to know the step by step operations of their client company, to whom they would develop the system or website. Interviews and documentation should be planned, scheduled and detailed out as possible to prevent further revisions on the developing software products. The more things to be left out and more revisions to be made, this would result to the delay of launching of the software products and higher development costs because developers will stay on the project longer. To prevent this to happen, always planned ahead and make an interview guidelines to make sure that nothing will be forgotten.

28. in how many ways can a code be formed if it must start and end with a even numbers​


The answer to this question will depend on how long the code is and what digits are allowed in the code.

Assuming the code is a 4-digit number (i.e., it has two digits between the even digits at the beginning and end), and that any digit from 0 to 9 is allowed in the code (i.e., digits can be repeated), the problem can be solved as follows:

There are 5 even digits (0, 2, 4, 6, 8) that can be used at the beginning and end of the code.

For each even digit chosen for the beginning, there are 5 even digits left to choose from for the end (since repetition is allowed).

Once the beginning and end digits are chosen, there are 10 digits (0-9) that can be chosen for each of the two middle digits (since repetition is allowed).

Therefore, the total number of possible codes is 5 x 5 x 10 x 10 = 2,500.

So there are 2,500 ways to form a code that starts and ends with an even number, with two digits in between.
