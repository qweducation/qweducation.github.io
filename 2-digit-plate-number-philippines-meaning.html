25. 5. If we subtract 4 from a digit at unit's place in a 2 digit number and add 4 to digit at ten's place, then resulting number is double. Find the number.


⇒  

: Let digit at unit's place = x

                Let digit at ten's place = y

                Original number = 10y + x

                After 4 is subtracted from digit at unit's place and 4 is added to digit at ten's place

                Let digit at unit's place = x - 4

                Let digit at ten's place = y + 4

                New number = 10(y + 4) + x - 4

                                         = 10y + 40 + x - 4

                                         = 10y + x + 36

                According to the given condition:

                10y + x + 36 = 2 (10y + x)

                 10y + x + 36 = 20y + 2x

                  10y - 20y + 36 = 2x - x

                   -10y + 36  = x

                   10y + x = 36

              So, original number = 10y + x = 36
