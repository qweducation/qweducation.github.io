C (n, r) = n! / (n-r)! r!, n = 12, r = 5 (since 6 + 4 + 2 = 12 and the commitee consists of 5 members.)

C (12, 5) = 12! / (12-5)! 5!

C (12,5) = 12! / 7! 5!

C (12, 5) = 479 001 600 / 5 040 (120)

C (12, 5) = 479 001 600 / 604 800

C (12, 5) = 792 ways

Therefore, there are 792 ways to choose 5 members for a commitee from 6 opposition senatorial candidates, 4 administration candidates, and 2 independent candidates.
