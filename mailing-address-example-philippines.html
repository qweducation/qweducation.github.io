9. ano ang ibig sabihin ng mailing address


Ang mailing address ang address na kung saan ay gusto mong tumatanggap ng mga padala sa iyo tulad ng sulat at mga packages.

Ang office address, residential address, at P.O box ay ilan sa mga pwedeng gamitin bilang isang mailing address.

Ngayon, ang tanong ay bakit hindi na lang ang address sa bahay o ang residential address ang gamitin ng lahat ng tao bilang mailing address nila, bakit ang iba ay gumagamit ng ibang address tulad ng P.O box?

Ang P.O box o post-office box ay isang locker sa Post Office na may numero. Kapag may nagpadala sa iyo at ginamit mo ang P.O box bilang iyong mailing address ay ilalagay ng postman ang padala sa iyo sa locker na iyon. Ngayon ikaw na ang pupunta sa post office upang kolektahin ang padala, na nakalagay sa iyong sariling P.O box. Ang P.O box ay nirerentahan at may bayad.

Ngayon, gumagamit ang ibang tao ng P.O box bilang kanilang mailing address dahil:

para sa privacy at security - kapag residential address kasi ang ginamit ay mas marami ang banta sa seguridad mo. Mas ligtas sa P.O box dahil doon ay may security measures na sinusunod. Hindi basta basta mananakaw o mawawala ang iyong package.

ito ay mas propesyunal - kapag may negosyo ka lalo na maliit pang negosyo ay mas mainam itong gamitin, para mas maganda ang impression ng kliyente

mas mabilis na delivery - mas mabilis makàrating ang package na pinadala sa receiver kung P.O box ang mailing address kumpara sa residential address

permanent mailing address - kapag ang receiver ay lumipat ng bahay ay di niya na kailangang baguhin pa ang kanyang mailing address kung P.O box ang gamit niya. Kapag residential kasi ang gamit na mailing address, kapag lumipat ng bahay ay dapat baguhin talaga ang mailing address dahil nga nagbago na ang residence ng receiver

19. Importance of mail merge with example


Mail merge is the process where you create personalized letters or address labels from a form letter by which is a document consisting of fixed contents same for each output. Its data source is typically a spreadsheet or a database which has a field or column for each variable in the template. When the mail merge is run, the software system creates an output document for each row in the database, using the fixed text exactly as it appears in the template, but substituting the data variable placeholders in the template with the values from the matching columns.  Mail merge also is often called print merge.

The advantages of mail merge is that the feature saves your time and efforts. Producing mass mailings is much simplified, especially compared to the process of preparing individual letters or envelopes to many people.  

The mail merge feature on mac is supported by several software titles, the Labels and Databases among them.

Use Mail Merge to add mailing addresses and other tailored information to your Publisher publications automatically.

Mail merge allows you to create a batch of customized documents for each recipient. A standard letter, for example, might be customized by addressing each recipient by name. The document is linked to a data source, such as a list, spreadsheet, or database. Merge fields are placeholders that tell Word where in the document to include data from the data source.

In Word, you work on the main document, inserting merge fields for the customized content you want to include. When the mail merge is finished, each name in the data source will receive a customized version of the merging document.
