23. If availed for y years, a premium for insurance costs Php 12,000 for the first year, and Php 15,000 for every succeeding year. Write a function which gives the total amount of premiums paid by the client.


Answer:

15000y - 3000

Step-by-step explanation:

In this question, we are given an unknown variable y. This is unknown because the client is free to choose the number of years.

Let's first focus on the statement "... Php 15000 for every succeeding years." This means that we have to multiply 15000 into y to find the total cost for an unknown number of years.

Thus, we have the expression 15000y.

However, our expression isn't complete. The given states that the 15000 doesn't apply for the first year, just 12000. This means that we need to subtract 1 from our variable.

Now, we have the expression 15000(y-1) or 15000y - 15000.

Finally, note that the first year of premium isn't free; it costs 12000. Because of this, we need to add 12000 to our expression.

We now have 15000y - 15000 + 12000. We just need to simplify this and we finally have the final answer.

Therefore, the answer to cost for an insurance premium is 15000y - 3000, where y is the number of years.
