24. 5. On March 15, 2020, the present value of ₱65,000 is ₱62,000 due on July 13, 2020. Find the discount rate. Solution: Given: Required: Formula: Computation: Conclusion:


Given:

Present value: ₱62,000Future value: ₱65,000Present date: March 15, 2020Future date: July 13, 2020

Required:

Discount rate

Formula:

Discount rate = (Future value - Present value) / Present value x (Days from present date to future date / 365)

Computation:

Discount rate = (₱65,000 - ₱62,000) / ₱62,000 x ((July 13, 2020 - March 15, 2020) / 365)

= (₱3,000) / ₱62,000 x (122 / 365)

= 0.04878 x 0.333

= 0.01628

= 1.628%

Conclusion:

The discount rate is 1.628%.
