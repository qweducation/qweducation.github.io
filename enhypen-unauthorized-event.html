23. how can data be secured to unauthorized and corrupted access?


Answer:

1 Keep Current on all Security Patches

2 Detect and Respond to Intrusions Quickly

3 IDS/IPS (Intrusion Detection System/Intrusion Prevention System)

4 Track records and provide reports on security-related events and incidents, such as failed and successful login attempts, malware activity or any other suspicious activity.

5 It’s crucial for all networks to use encryption if they store privacy-protected data. This applies to both the connections made by authorized users from outside the data center to access the system and network links between nodes in a multi-server system.

You can use a VPN layer between the users and the system or implement an SSL/TLS to encrypt network traffic. Inside the system, communications can be secured using IPsec, SSL/TLS, or some other VPN technology.

6. Encrypt Data-at-Rest

Encryption of data at rest ensures that data is stored securely and not as plain text. As data is written to the disk, it is encrypted via a set of secret keys which is known only to authorized administrators of the system.

It protects a computer network from unauthorized access - The answer is : Firewall.

A firewall is a software that normally monitors the activity of the network. It will prompt for an incoming and outgoing of packets from and to the network. Network administrators normally assigned rules on the network using the firewall, by allowing or blocking applications to use internet and browsers in sending and receiving packets of data. By this, it will minimize unwanted attacks in the network or interception of data in the network. When the firewall detects an attack, it would prompt you automatically and blocked the application which suspecting the attack is coming from.
