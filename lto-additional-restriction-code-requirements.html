20. A 5-digit pin code cannot begin with zero and the remaining digits have no restrictions. if repetition of digits is allowed, then what is the probability that the pin code begins with a 7 and ends with an 8?


Answer:   [tex]\frac{1}{90}[/tex]

Explanation:

Step 1: Determine the over-all number of possibilities.

Since a 5-digit PIN code cannot start with zero, therefore the first digit would be restricted by one less digit, and then the rest would be composed of any of the numerical digits from 0 to 9.

Let N be the number of possible PIN number outcomes. Therefore:

N = 9*10*10*10*10

N = 90000 PIN Numbers

Step 2: Determine the number of possible outcomes

To determine the number of possible outcomes, we must identify the constraints first. The constraint is that the pin code should begin in 7 and end at 8, therefore, we have to lock them in on the first and last position. So, the first and last slot should have no other possibility but 7 and 8 in the first and last digit, respectively. This way, we guarantee that the second, third, and fourth digits are the only digits have to worry about the number of possible PIN codes:

Let M be the number of possible codes that starts with 7 and ends with 8. Therefore:

M = 1 * 10 * 10 * 10 * 1

M = 1000 PIN numbers

Step 3: Finalize.

Then, we have to put the two numbers in rational form. Out of 90,000 PIN numbers that are available, there are only 1000 which starts at 7 and ends with 8. In numeric form, it looks like:

P = [tex]\frac{1000}{90000}[/tex]

Then, switch them to lowest terms:

P = [tex]\frac{1}{90}[/tex]

Therefore, there is a [tex]\frac{1}{90}[/tex] chance that the PIN code begins with a 7 and ends with an 8.
