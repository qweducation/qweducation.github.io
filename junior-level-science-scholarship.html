7. Input a grade level (Freshman, Sophomore, Junior, or Senior) and print the corresponding grade number [9-12]. If it is not one of those grade levels, print Not in High School.


Answer:

If the input is "Freshman", print "9".

If the input is "Sophomore", print "10".

If the input is "Junior", print "11".

If the input is "Senior", print "12".

If the input is none of the above, print "Not in High School".

Here is an example implementation in Python:

grade_level = input()

if grade_level == "Freshman":

 print(9)

elif grade_level == "Sophomore":

 print(10)

elif grade_level == "Junior":

 print(11)

elif grade_level == "Senior":

 print(12)

else:

 print("Not in High School")

Explenation:

This code first takes in the input 'grade_level', and then uses an if-else statement to check the value of 'grade_level'. If it is "Freshman", "Sophomore", "Junior", or "Senior", the corresponding grade number is printed. Otherwise, the code prints "Not in High School".
