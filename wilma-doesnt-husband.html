Further explanation:

CONDITIONALS are statements expressing the result of a given condition. The main clause describes the outcomes, whereas the if- and when clause states the condition. Both positive and negative sentence patterns are capable of containing conditional assertions. Conditionals may come before the main sentence of a statement. If the main clause appears before the conditional, the meaning of the statement remains the same.

Zero conditional

Zero conditional is concerned with customs and universal truths like laws. In order to build it, use the following syntax: if/when+ present simple >> present simple condition main clause.

First conditional

The first conditional is used to discuss future events that are conceivably or provably going to occur. The most frequent version of this type utilizes the following syntax: if/when + present simple >> will + infinitive condition main clause. It can also be made using the following types: unless, as long as, as soon as, or in case.

Third conditional

When dealing with situations that are impossible in the present and unlikely to occur in the future, the second conditional is utilized. It is made using the following structure: main clause: if + past simple >> would + infinitive.

Fourth conditional

When dealing with events or changes in a previous scenario that lead to various outcomes, the third conditional is utilized. It has the following structure: If followed by would have followed by the past participle forms the main phrase.
