23. The voltage V(in volts) coming from an electricity distributing company is fluctuating according to the function V(t)= 150+110sin(150rt) at time t in seconds. Determine the first time it takes to reach 220 volts.


Answer:

To solve this problem, we need to find the value of t such that V(t)=220. Substituting the given expression for V(t) into this equation, we have:

220 = 150 + 110 * sin(150rt)

We can rearrange this equation as follows:

sin(150rt) = (220-150)/110

Since the sine function has a range of [-1, 1], the value on the right-hand side of this equation must be between -1 and 1. We can check that this is the case:

(220-150)/110 = 0.73

Therefore, the equation has a solution. To find the value of t, we can use the inverse sine function, denoted as "sin^-1" or "arcsin". The inverse sine function returns an angle (in radians) whose sine is equal to the input value. In other words, if x = sin^-1(y), then y = sin(x).

Thus, we can find t as follows:

t = (1/150r) * sin^-1((220-150)/110)

This is the first time it takes to reach 220 volts.

Note: If you want the time in seconds, you need to use the conversion factor 1 minute = 60 seconds.
