In addition to decimals, here's how to add two or more decimals.

Decimal addition works similarly to integer addition. Convert them to similar decimal numbers and align the numbers vertically, one above the other, so that the decimal point is exactly on the vertical line. Add as usual, as we learned with integers.

The rules for adding decimals are:

Write the digits of a given number up and down so that all decimal points are on the same vertical line.

Add like you would add an integer.Place the total decimal point vertically below the other decimal points.

Subtracting Decimals

To subtract, follow the same method. Align the decimal points and then subtract.
