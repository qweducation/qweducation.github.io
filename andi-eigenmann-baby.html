21. Andy, Ben, and Chris donated P550 altogether. Ben donated P100 more than Andy. Chris donated 3 times than Andy. Find how much did each of them donated.please help!​


Answer:

Andy donated P90

Ben donated P190

Chris donated P270

Step-by-step explanation:

a + b + c = 550

a = ?

b =  a + 100

c = 3a

solution:

a + a + 100 + 3a = 550

simplify:

5a + 100 = 550

subtract 100 from both sides:

5a + 100 - 100 = 550 - 100

5a = 450

Divide both sides by 5:

[tex]\frac{5a}{5}[/tex] = [tex]\frac{450}{5}[/tex]

a = 90

b = (90) + 100 = 190

c = 3(90) = 270
