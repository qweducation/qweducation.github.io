8. A plumber charges 55 for a service call plus 35 per hour. If the plumbers bill was 160, how many hours was he there


Answer:

3 hours

Step-by-step explanation:

Let x be the number of hours

35x + 55 = 160 (Find x)

Answer:

The plumber worked for 3 hours.

Step-by-step explanation:

Flat/Basic charges:  55

Additional charges: 35 per hour

Hour: x

Total Bill: f(x) = 160

Equation:

f(x) = 55 + 35x

Solve for number of hours, x:

160 = 55 + 35x

35x = 160 - 55

35x/35 = 105/35

x = 3

The plumber worked for 3 hours.

17. What amount should be paid on January 13, 2016, for a loan of 16,000 pesos made on July 7, 2015, at 8% simple interest using ordinary interest using approximate time?.


The simple interest using ordinary interest using approximate time is ₱675.56

The interest due at the end of a term is known as simple interest. Only the principal (original value) earns interest in a simple interest transaction. The principal can be either the amount deposited in an account or the amount borrowed. The approximate time is calculated in the same way as the actual time, but with the assumption that each month has 30 days.

First, we must determine the exact and approximate times.

The actual time is the number of days that pass between two dates.Approximate time - is the number of days assuming that each month has 30 days.

The actual time span is 190 days, from July 7, 2015, to January 13, 2016. However, because there are 6 months and 6 days between, the approximate time is 186 days (#days = (6 x 30) + 6).

Meanwhile, an ordinary interest rate uses 360 days per year, whereas an exact interest rate uses 365 days (366 days if it is a leap year) per year.

We use the following formula to solve for the interests:

I = P x R x T

where,

I = InterestP = Principal start amount (which is ₱16,000)R = Interest Rate (which is 8% = 0.08)T = Time (in years or in a fraction of a year)

a) ordinary interest applied to the actual time

I = P x R x T

= ₱ 16,000 x 0.08 x (190/360)

= ₱ 675.56

b) ordinary interest applied to the approximate time

I = P x R x T

= ₱ 16,000 x 0.08 x (186/360)

= ₱ 661.33

c) exact interest applied to the actual time

I = P x R x T

= ₱ 16,000 x 0.08 x (190/365)

= ₱ 666.30

d) exact interest applied to the approximate time

I = P x R x T

= ₱ 16,000 x 0.08 x (186/365)

= ₱ 652.27

The following equation can be used to calculate the interest rate (R):

R = I / (P x T)

= ₱ 2,100 / (₱ 12,000 x 2.5)

= 0.07 or 7%

28. On July 1, 2015, Hanson Company borrowed 200,000 foreign currencies (FC's) from a foreign lender evidenced by an interest-bearing note due on July 1, 2016. The note is denominated in foreign currencies. The Philippine peso equivalent of the note principal is as follows: Date Amount July 1, 2015 (date borrowed) P1950.000 December 31, 2015(Hanson’s year end) 220,000 July 1 2016 (date repaid) 230,000 In its 2016 income statement, what amount should Hanson include as a foreign exchange gain or loss on the note? A. P35,000 gain B. P35,000 loss C. P10,000 gain D. P10,000 loss show solution& short explanation


Answer:

brainliest please

Explanation:

Solution:

To determine the foreign exchange gain or loss on the note, we need to compute for the difference between the peso amount paid upon repayment of the note and the peso value of the note when it was borrowed.

Peso value of the note when borrowed = P1,950,000

Peso value of the note when repaid = P230,000

Foreign exchange gain/loss = P230,000 - P1,950,000 = (P1,720,000)

Since the foreign exchange gain/loss is negative, it means that Hanson has a foreign exchange loss of P1,720,000. However, this amount needs to be adjusted for the interest expense incurred on the note.

Interest expense = P220,000 x 12/12 x 6% = P13,200

Adjusted foreign exchange loss = P1,720,000 - P13,200 = (P1,706,800)

Therefore, the answer is B. P35,000 loss.

Short Explanation:

To compute for the foreign exchange gain or loss on a note denominated in foreign currencies, we compare the peso value of the note when it was borrowed and when it was repaid. If the peso value when repaid is higher than when borrowed, there is a gain. If it is lower, there is a loss. We also need to adjust for any interest expense incurred on the note.
