2. flowchart or pseudo code that the sum of number 1 up to 1000. Print the value of the counter and the output in each time it enters the loop.


The question says that it has a loop so this is already a clue as to how we should go about making the pseudo code. I usually see two kinds of loops "for", and "while".

First let's try the pseudo code using a for loop:

I'll be putting the actual pseudo code in underline, with explanations in normal font.

Before anything let's initialize our variables first. We need a counter, and a sum, right? So, let's set the following variables:

start

counter = 1, since we start counting at 1

sum = 0, since we haven't started adding yet.

Now, let's go make use the for loop:

for (counter = 1, counter <1001, counter = counter + 1)

The first two arguments say that the loop will continue starting from counter = 1, up to counter <1001 (or <=1000). The third one says that after every iteration (every time it finishes one round of the commands inside the loop), we will add 1 to the counter. This is important for the counter to actually count, and not just stay as 1 forever.

{

sum = sum + counter; This adds the counter to the original sum. We initialized sum to be 0, so in the first iterate, sum = 0+1 = 1.

print (counter); Here, the counter is still 1, so it prints 1

print(sum); Here, sum is already 1 from the sum = sum + counter line, and it prints that too.

}

end

Now what happens during the second iteration:

counter = 2 (from counter = counter + 1)

sum = 1 + 2 = 3 (from previous sum=1 and new counter =2)

And this continues on until 1000.


Here is an example for the while loop, using the same logic as the one in the for loop:

start

counter = 1

sum = 0

while (counter <1001){

sum = sum + counter;

print (counter);

print(sum);

counter = counter + 1;

}

Answer:

1. 8

2. 16

3. 120

Step-by-step explanation:

1. (a) Because It is tossing a coin three times. Each time there will be 2 situations Then the total number of outcomes should be 2*2*2=8. {MULTIPLICATION}

(b) Because It is answering two multiple questions with four choices each. For the first question, there is 4 situations. The second question. It is also 4 situations. Then the total number of outcomes should be 4x4=16. {MULTIPLICATION}

2. Because the numbers should be formed from 1,3,5,7,9, 5 numbers. Then the picking ways should beC4,5=5!/4! 1! = 5. Then we arrange the 4 numbers, the arranging ways are 4! = 4x3x2x1 =24.

Hence the total number of outcomes should be 5x24 = 120.
