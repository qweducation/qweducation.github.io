Answer:h = 7.94m

Step-by-step explanation:

The length of the brace is given by

L = square root of H² + W²

L = 12m

fixing distance = 9m

solve for H:

Apply exponents

L² = W² + H²

subtract W²

L² - W² = H²

12² - 9² = H²

144 - 81 = H²

square root of 63 = H²

taking square root

square root of 63 = H²

H = 7.94
