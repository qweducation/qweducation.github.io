A function f is 1 -to- 1 if no two elements in the domain of f correspond to the same element in the range of f .

In other words, each x in the domain has exactly one image in the range. ... If no horizontal line intersects the graph of the function f in more than one point, then the function is 1 -to- 1[tex].[/tex]
